\chapter*{Abstract}
\label{chap:abstract}
\addcontentsline{toc}{chapter}{\nameref{chap:abstract}}

% Persona Engine is an Unreal Engine plugin that allows users to generate game characters from natural language descriptions. Current character creation systems require users to manually adjust a large number of numerical parameters, which is time-consuming and technically demanding. This project replaces that process with a text-based input system. The user provides a plain-language description of a character, and the system uses a large language model to produce a structured JSON output containing normalized values for physical appearance parameters and personality traits. The appearance parameters are applied directly to Unreal Engine's MetaHuman morph targets. The personality parameters, based on the Big Five model, are used to generate context-appropriate dialogue and influence in-game behavior. The result is a unified pipeline that produces both the visual representation and behavioral profile of a character from a single text input.
